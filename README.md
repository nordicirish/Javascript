<h2>FizzBuzz</h2> 
Uses a while loop to output an array of numbers between 1 and 100.
<br>
<ul>
<li>Numbers divisible by 3 are output as the string "Fizz".</li>
<li>Numbers divisible by 5 are output as the string "Buzz"</li>
<li>Numbers divisible by 3 and 5 output as the string "FizzBuzz".</li>
</ul>  

<h2>Reaction Tester</h2> 
Simple game to user test reaction speed 
<br>
<ul>
<li>Generates circles or square shapes </li>
<li>Shapes have random colours, sizes and positions</li>
<li>Shapes are generated at random time intervals</li>
<li>Displays the time taken for the user to click on a shape after it appears</li>
</ul>
